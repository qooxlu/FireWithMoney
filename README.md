## English Version

### Description

Automatically purchase and load bullets from bank balance or backpack cash when reloading if no ammo in inventory.

### Core Features

- Press **Shift+B** to toggle payment mode (BankğŸ’³ / CashğŸ’µ)
- Press **R** to reload or **T** to switch ammo type
- Display all ammo types with tier-based sorting
- Auto-refund on purchase failure
